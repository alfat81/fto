# Фабрика торгового оборудования — ФТО

Сайт фабрики торгового оборудования в Нижнем Новгороде. Производим стеллажи, витрины, кассовые боксы, кресла. Полный цикл: проектирование, изготовление, монтаж.

## Технологии
- **Фронтенд:** HTML5, CSS3, JavaScript (vanilla, без сборщика)
- **Бэкенд:** Node.js, Express, Nodemailer (отправка заявок на email)
- **Хостинг фронтенда:** GitHub Pages (директория `/docs`)
- **Хостинг бэкенда:** Render.com
- **SMTP:** Mail.ru (российский email-провайдер)

## Структура сайта

| Страница | Назначение |
|----------|------------|
| `index.html` | Продающая главная (hero, категории, услуги, кейсы, отзывы, FAQ, форма расчёта) |
| `catalog.html` | Каталог товаров с фильтрами и поиском |
| `services.html` | Услуги (7 шагов под ключ: замер → 3D-проект → производство → монтаж) |
| `cases.html` | Портфолио реализованных проектов |
| `about.html` | О фабрике (история, производство, команда, сертификаты) |
| `delivery.html` | Доставка и оплата |
| `contacts.html` | Контакты + Яндекс.Карта + форма обратной связи |
| `privacy.html` | Политика конфиденциальности (152-ФЗ) |

## Настройка

### Фронтенд (GitHub Pages)
1. Файлы фронтенда в папке `docs/`
2. В настройках GitHub Pages: Branch `main`, Folder `/docs`
3. URL: `https://alfat81.github.io/fto/`

### Бэкенд (Render.com)
См. подробную инструкцию в [backend/README.md](backend/README.md).

После деплоя бэкенда укажите его URL в `docs/js/config.js`:
```js
backend: {
    url: 'https://fto-backend-xxxx.onrender.com'
}
```

## Что ещё нужно сделать (вопросы к заказчику)

Сайт содержит плейсхолдеры `[УТОЧНИТЬ]` / `[Подставить реальные данные]` в следующих местах:

1. **ИНН и ОГРН** — в footer всех страниц и в privacy.html
2. **Год основания и количество проектов** — в hero и about.html (использованы предположения: 2010, 500+)
3. **Кейсы** — названия клиентов и фото объектов (использованы вымышленные кейсы как примеры структуры)
4. **Отзывы** — реальные отзывы с фото клиентов
5. **Логотипы клиентов** — реальные логотипы вместо текстовых заглушек
6. **Сертификаты** — номера и сканы PDF
7. **Соцсети** — реальные ссылки ВК и WhatsApp (использованы примеры)
8. **Карта** — координаты в Яндекс.Карте могут потребовать уточнения
