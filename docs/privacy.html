Политика конфиденциальности

Дата вступления в силу: 29 июня 2025 г.

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей сайта alfat81.github.io/fto (далее — «Сайт»), принадлежащего Фабрике торгового оборудования «ФТО» (далее — «Оператор»).

Обработка персональных данных осуществляется в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

1. Общие положения

Персональные данные — любая информация, относящаяся к прямо или косвенно определённому физическому лицу (субъекту персональных данных).

Оператор — Фабрика торгового оборудования «ФТО», осуществляющая обработку персональных данных через сайт.

Использование Сайта означает безоговорочное согласие пользователя с настоящей Политикой и указанными в ней условиями обработки персональных данных.

2. Состав персональных данных

Оператор обрабатывает следующие персональные данные, предоставляемые пользователем добровольно через формы Сайта:

  • Форма обратной связи (страница «Контакты»): имя, телефон, email, текст сообщения.
  • Форма оформления заказа (корзина): имя, телефон, адрес доставки.

Оператор не обрабатывает специальные категории персональных данных (раса, национальность, политические взгляды, религиозные убеждения, состояние здоровья, интимная жизнь).

3. Цели обработки

Персональные данные обрабатываются исключительно в следующих целях:

  • Обработка заявок и заказов, оставленных через Сайт;
  • Связь с пользователем для уточнения деталей заказа и расчёта стоимости;
  • Информирование о статусе заказа;
  • Ответ на вопросы, заданные через форму обратной связи.

Обработка персональных данных для иных целей не производится. Оператор не передаёт персональные данные третьим лицам для маркетинговых или рекламных рассылок без отдельного согласия пользователя.

4. Правовое основание обработки

Обработка персональных данных осуществляется на основании:

  • Согласия пользователя на обработку его персональных данных, выраженного путём установки отметки в специальном чекбоксе при заполнении форм Сайта;
  • Необходимости исполнения договора, стороной которого является пользователь (для обработки заказов);
  • Требований законодательства РФ.

5. Сроки обработки и хранения

Персональные данные хранятся в течение срока, необходимого для достижения целей обработки, но не более 3 (трёх) лет с момента последнего взаимодействия с пользователем, если иной срок не предусмотрен законодательством РФ.

По истечении указанного срока персональные данные уничтожаются либо обезличиваются.

6. Передача данных третьим лицам

Оператор не передаёт персональные данные пользователей третьим лицам, за исключением случаев:

  • Когда это необходимо для исполнения заказа (например, передача контактных данных в службу доставки);
  • Когда это требуется законодательством РФ (по запросу уполномоченных государственных органов).

Заказы, оставленные через Сайт, передаются ответственным сотрудникам Оператора через внутренний канал связи (Telegram-бот), доступ к которому ограничен.

7. Меры защиты персональных данных

Оператор принимает необходимые организационные и технические меры для защиты персональных данных от неправомерного доступа, изменения, раскрытия или уничтожения:

  • Ограничение доступа к персональным данным только для сотрудников, нуждающихся в них для выполнения должностных обязанностей;
  • Использование защищённых каналов связи (HTTPS) для передачи данных;
  • Хранение данных на серверах, расположенных в Российской Федерации.

8. Права субъекта персональных данных

Пользователь имеет право:

  • Получать информацию об обработке своих персональных данных;
  • Требовать уточнения, блокирования или уничтожения своих персональных данных;
  • Отозвать согласие на обработку персональных данных, направив заявление на электронный адрес Оператора;
  • Обжаловать действия или бездействие Оператора в уполномоченном органе по защите прав субъектов персональных данных (Роскомнадзор).

9. Cookies

Сайт использует технологию localStorage браузера для хранения содержимого корзины между сессиями. Эта информация хранится локально в браузере пользователя и не передаётся на сервер Оператора. Пользователь может очистить localStorage через настройки браузера в любой момент.

Сайт не использует cookies для отслеживания поведения пользователя.
