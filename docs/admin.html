Как пользоваться: загрузите существующие товары кнопкой «Импорт JSON» (можно загрузить несколько файлов сразу) или начните с нуля. Изменения сохраняются в браузере (localStorage). По готовности нажмите «Скачать все товары» — получите набор JSON-файлов для копирования в docs/data/products/. Не забудьте обновить index.json — он скачается автоматически в архиве.

Новый товар

Уникальный идентификатор. Используется в имени файла (5.1.json) и для изображения (5.1.jpg).
Если заполнить, цена покажется со скидкой.
Можно оставить пустым — будет использоваться images/{ID}.jpg
Пары «ключ — значение», например: «Высота — 1800 мм», «Нагрузка — 150 кг».
